# R Mastery Path

A free, structured, module-wise R training program — entry level to master level, with runnable code in every module and a hands-on project at the end of each level. Focused on R's real-world niche: data analysis and statistics.

**Live site:** https://sigilipelli.github.io/r-mastery-path/

## Contents

- **Level 1 · Entry** — language foundations, vectors, data frames, basic plotting
- **Level 2 · Intermediate** — dplyr, data cleaning, ggplot2, statistics basics
- **Level 3 · Advanced** — statistical modeling, R Markdown, Shiny, APIs
- **Level 4 · Master** — production R (plumber), Shiny at scale, big data, CRAN publishing

## Local development

```bash
python3 -m venv .venv
.venv/bin/pip install mkdocs-material
.venv/bin/python -m mkdocs serve
```
